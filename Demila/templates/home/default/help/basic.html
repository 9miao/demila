如何设置文章格式？

你可以使用HTML标签编辑你在{$domain}的评论和文章。

强调文本

用<strong>和</strong>或<em>和</em>包裹强调文本，比如:

粗体文本 <strong>粗体文本</strong>

斜体文本 <em>斜体文本</em>

链接

格式为<a href="URL">链接显示文本</a>，比如你要把某文本链接到{$domain}，你可以这样写:

<a href="http://{$domain}/">链接到{$domain}</a>

链接到{$domain}

图像

首先复制图像的链接地址，然后把链接地址写到URL的位置，<img src="URL">，比如我们网站LOGO的地址为http://{$domain}/static/logo/logo-dark.png于是你可以这样书写:

<img src="http://{$domain}/static/logo/logo-dark.png">

引用

<blockquote>引用文字</blockquote>。

创建列表

格式如下：

不带序号列表 <ul> <li>列表条目</li> <li>列表条目</li> </ul>

  • 列表条目
  • 列表条目
带序号列表 <ol> <li>列表条目</li> <li>列表条目</li> </ol>
  1. 列表条目
  2. 列表条目

更多关于如何编辑列表的详情

标题

格式如下:
一级标题
<h3>一级标题</h3>
二级标题

<h4>二级标题</h4>

代码

简单代码请使用<code>和</code>，大量代码请使用<pre>和</pre>换行，比如:<code>小段多行代码</code>

普通文本换行

普通文本换行请使用<br />标签。
